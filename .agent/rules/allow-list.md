# Terminal Command Allow List

This file contains commands that are explicitly safe to auto-run.
Even if a command is not here, it will be auto-run unless it matches the Deny List.
*Note: You can request to add any command to this list by asking me.*

- `git status`
- `git status --ignored`
- `git status --porcelain=v1`
- `ls`
- `dir`
- `pwd`
